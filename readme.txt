You can easily add the TinyMCE (WYSIWYG) editor to WooCommerce product categories and tags "description" for better formatting options.

== Description ==
This plugin enables the TinyMCE (WYSIWYG) editor for WooCommerce product categories and tags "description". This lets you easily add text formats like H1, and H2 titles, bullet points, images, and bold text. This would be great for SEO purposes.

== Installation ==
###Automatic Installation from the WordPress Dashboard

1. Login to your admin panel
2. Navigate to Plugins -> Add New
3. Search **Term Description: Rich Text Editor for WooCommerce**
4. Click install and activate respectively.

###Manual Installation from the WordPress Dashboard

If your server is not connected to the Internet, then you can use this method

1. Download the plugin. A ZIP file will be downloaded.
2. Login to your site\'s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click choose file, select the plugin file and click install

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use
this method

1. Download the plugin. A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favorite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you
   can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin

== Frequently Asked Questions ==
### How do I enable the TinyMCE editor for product category descriptions?

Once the plugin is activated, the TinyMCE editor will automatically replace the default textarea in the product category description field. Simply go to **Products > Categories** in your WordPress dashboard, and you'll see the enhanced editor available for use.

### Can I use this plugin with custom taxonomies?

No, the plugin is designed primarily for WooCommerce product categories and tags.

### Does this plugin affect the performance of my site?

The plugin is lightweight and optimized for performance. It only loads the TinyMCE editor when you're editing product category descriptions, so it shouldn't significantly impact your site's loading speed or performance.

### What formatting options are available with the TinyMCE editor?

The TinyMCE editor offers a wide range of formatting options, including:
- Bold, Italic, Underline
- Bullet Points and Numbered Lists
- Links and Anchor Texts
- Blockquotes
- Text Alignment
- Adding Media (images, videos, etc.)

### How does the plugin affect SEO?

By allowing richer formatting and better-organized content, the plugin can help improve the readability and SEO of your category descriptions. However, the actual impact on SEO will depend on the content you create and how well it aligns with SEO best practices.

### Is the plugin compatible with all WordPress themes?

The plugin is designed to be compatible with most WordPress themes, particularly those that support WooCommerce. However, if you encounter any compatibility issues, please reach out to our support team.

### Can I revert to the default WooCommerce editor after installing this plugin?

Yes, if you prefer to use the default WooCommerce editor, you can simply deactivate the plugin, and your product category description fields will revert to the standard WooCommerce text editor.

### Is this plugin regularly updated?

Yes, we regularly update the plugin to ensure compatibility with the latest versions of WordPress and WooCommerce, as well as to introduce new features and improvements.
